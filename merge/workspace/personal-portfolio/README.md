# Personal Portfolio Website

A modern personal portfolio website built with Next.js, React, and Tailwind CSS.

## Features

- Responsive design that works on all devices
- Clean, modern UI with smooth animations
- Sections for about, skills, projects, and contact
- Easy to customize with your own content
- Built with TypeScript for type safety

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

To customize this portfolio with your own information:

1. Edit `app/page.tsx` to update your name, bio, skills, and projects
2. Replace the placeholder avatar with your own image
3. Update the social media links
4. Add your own project images

## Deployment

This project is ready to deploy to Vercel or any other Next.js hosting platform.

```bash
npm run build
npm run start
```
